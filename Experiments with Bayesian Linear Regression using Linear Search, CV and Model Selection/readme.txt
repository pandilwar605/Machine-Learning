The code contains only one file linear_regression.py which has seperate function for each tasks.
All data files are in pp2data folder.
code can be run by using following command:
python linear_regression.py train-1000-100.csv trainR-1000-100.csv test-1000-100.csv testR-1000-100.csv
and similarly for other datasets by just changing names

Program will generate graphs for all the tasks based on filename given.
